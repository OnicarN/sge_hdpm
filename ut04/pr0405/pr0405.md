use map y reduce para obtener la
frecuencia de cada palabra.

y supón que no hay símbolos de puntuación.

```python

texto = "Hola hola mundo mundo cruel"

def frecuencia (texto):
    espacios = texto.lower().split(" ")
    from functools import reduce
    mapeado = list(map(lambda a: (a,espacios.count(a)),espacios))
    dic = dict(mapeado)
    print(dic)
frecuencia (texto)
```

 crear una nueva lista con las cadenas que tengan
   más de tres letras y en las que todas las letras
