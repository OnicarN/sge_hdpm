6.- Combinar dos diccionarios
Escribe un programa que tome dos diccionarios de productos y precios,
 y combine los productos comunes sumando sus precios, sin duplicar los elementos nicos.
'''

```python
productos1= {"Nintendo":250,"Play":823}
productos2 = {"Nintendo":250,"Xbox":909}


nuevoDiccionario = {}
for j,k in productos1.items():
    for l,m in productos2.items(): 
        if j == l :
            nuevoDiccionario[j] = k + m
        else:
            nuevoDiccionario[j] = k
            nuevoDiccionario[l] = m    
print(nuevoDiccionario)
```       

5.- Diccionario invertido
Escribe una función que tome un diccionario y devuelva otro con 
las claves y valores intercambiados
(lo que antes eran valores ahora son claves, y viceversa).
'''
```python
diccionario = {"Nombre":"Dani"}

def voltear(dict):
    diccionario = {}
    for k,v in dict.items():
        diccionario[v] = k
    return diccionario     
res  = voltear(diccionario)
print(res)
```

Contador de frecuencias de palabras
Escribe un programa que tome una frase y use 
un diccionario para contar la frecuencia de cada palabra.
'''
```python
fraseUsuario = input("Introduce frase: ")
diccionario = {}
listaEspacios = fraseUsuario.split()
print(listaEspacios)

contadorRepetidos = 0
for palabra in listaEspacios:
    contadorRepetidos = listaEspacios.count(palabra)

    diccionario[palabra] = contadorRepetidos
print(diccionario)
```
